Soar Capture Dataset Instruction and Directory

Introduction:

This is a dataset of Soar Capture.

./Captures contains all the data we captured by Soar Capture Suite as well as outputs data exported by Soar Capture Suite software.

Format of each data piece:

Each piece of capture data is organized to a specific folder. The name of the data piece folder contains abstract information of the data piece.

The format is presented as {Name}-{Action}({More action detail})-{Capture count}-{Optional Tag}

  • Example 1: Jeffrey-Walk(Moving)-00-B means this piece of data is the first(00) capture for Jeffrey, performing a moving walking action. "-B" means the exported data is somehow broken and is not readable for Soar Viewer.
  • Example 2: Joseph-Idle(TPose)-01 means this piece of data is the second(01) capture for Joseph, performing a T-Pose Idle action.

The data in each folder has random name that does NOT mean anything but also not changeable unless to export them again with desired names. Below is an example of how the format of the file matters:

  • {Random}.srd: Soar capture raw data.
  • {Random}.tml: Soar capture raw data.
  • {Random}.mp4: Exported texture for each frame.
  • {Random}.sgv: Exported vector graphics.
  • {Random}_geometry.m3u8: Exported media file for geometry.
  • {Random}_master.m3u8: Exported master media file for SDK and software to load and play the capture.
  • {Random}_texture.m3u8: Exported media file for texture.

Dataset directory and category explanation:

  • ./Captures/Full - Captures that focusing on the full body of participants
    • ./Captures/Full/.Idle - All full body captures for idle action
    • ./Captures/Full/.Jump - All full body captures for jumping action
    • ./Captures/Full/.Run - All full body captures for running action
    • ./Captures/Full/.Walk - All full body captures for walking action
  • ./Captures/Upper contains
    • ./Captures/Upper/.Face - All upper body captures focusing on face detail
    • ./Captures/Upper/.Idle - All upper body captures for idle action
    • ./Captures/Upper/.Wave - All upper body captures for waving action
